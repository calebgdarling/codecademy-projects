HTML Cheatsheet | Tables

HTML Tag Title Description
<table> </table> Table Tags The table tags are used to create a new table. All your columns & rows are held between these opening and closing tags.
<tr> </tr> Table Rows Table row tags will create a new row in your table.
<td> </td> Table Data The table data tags are nested within each table row. These will create columns (or cells if your prefer).
<th> </th> Table Headers These will create table headers. These behave the same as table data tags but will bold & center the text by default.
